The foundation of this project was created as part of an AutoCAD Architecture implementation study involving several members of a large architectural firm, representatives from Autodesk, Inc. and a few independent consultants. The exercise lasted for one week. The first two days consisted of training on several AutoCAD Architecture features. The remaining three days were spent as a design charette where members of the architectural firm used the software to design and present the building.

 

This project has been further refined by Autodesk, Inc. to present some of the new features available within AutoCAD Architecture, and to offer an example of a completed project within AutoCAD Architecture.
